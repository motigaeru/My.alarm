# アラームアプリ

制作期間4ヶ月

##ツールのバージョン
- X Code 15.0.1
- VS Code 1.59.1
- Android Studio Android Studio 2022.1, build AI-221.6008.13.2211.9619390. Copyright JetBrains s.r.o., (c) 2000-2023

# 機能

- アラーム設定機能
- アラーム音を消す機能
- アラームに名前をつける機能
- バイブレーション機能
- スヌーズ機能(５分刻み)
- 鳴る曜日を指定できる機能
- 音量調節機能
- 通知機能（バックグランド）
