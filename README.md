# アラームアプリ

制作期間4ヶ月

# 機能

- アラーム設定機能
- アラーム音を消す機能
- アラームに名前をつける機能
- バイブレーション機能
- スヌーズ機能(５分刻み)
- 鳴る曜日を指定できる機能
- 音量調節機能
- 通知機能（バックグランド）
